# 100 Days Code Practice — Python, PHP & JavaScript

Este repositorio contiene mis ejercicios diarios de práctica para mejorar mi base de código en **Python**, **PHP** y **JavaScript**.

El objetivo principal de este reto es practicar durante 100 días, comenzando desde ejercicios básicos con funciones y avanzando poco a poco hacia problemas más complejos relacionados con desarrollo web, APIs, validaciones, manejo de datos, lógica de negocio y buenas prácticas de programación.

---

## Objetivo del reto

Mejorar mi capacidad para escribir código más limpio, reutilizable y estructurado en tres lenguajes importantes para mi carrera como desarrollador web:

- Python
- PHP
- JavaScript

Durante este reto voy a practicar:

- Funciones
- Parámetros
- Retorno de valores
- Validaciones
- Manejo de strings
- Manejo de arrays/listas
- Objetos y diccionarios
- Formateo de datos
- Lógica de negocio
- Simulación de APIs
- Procesamiento de JSON
- Buenas prácticas de código
- Refactorización
- Resolución de problemas
